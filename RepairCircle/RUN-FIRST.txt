Run in PowerShell from this folder:

1) dotnet restore
2) dotnet build
3) dotnet ef database update
4) dotnet run

If you previously used this exact package name and want a clean database:

 dotnet ef database drop --force
 dotnet ef database update
 dotnet run

Open:
 https://localhost:7228
 or
 http://localhost:5107

Notes:
- This project uses local login and registration only.
- External login providers are intentionally not configured.
- Password reset email sending is not configured.
